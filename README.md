# 算法分类
按照算法内容，基础算法的类型分为：
- 数组
- 字符串
- 链表
- 二叉树
- 栈 & 队列
- 哈希
- 查找
- 排序
- 其他（回溯法、动态规划&贪婪算法、位运算、其他类型）
